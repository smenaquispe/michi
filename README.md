# Michi (tik tak toe)
This is a game of tik tak toe made with react and javascript

## Technologies:
- Reactjs
- Javascript
- React module css
- Motion
- Tabler Icons

## Run the project
you just have to follow this steps:
- Go to `/app` directory 
- Run `npm install` 
- Wait to install the whole list of packages
- Run `npm run dev`
- Just play
